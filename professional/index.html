Rocket League Sideswipe

Android | iOS

Rocket League Sideswipe is a version of the popular multiplayer car soccer title Rocket League, but reimagined for mobile. I worked on the Demiurge Studios engineering team, which Psyonix contracted to help deliver the title on mobile devices. I primarily acted as a UI engineer, implementing new UI screens and optimizing previously created screens. I also helped with a variety of other systems, including:

  • Texture streaming fixes for in-game objects and UI
  • Memory and performance optimization
  • Analytics
  • Metagame features
  • Startup flow
  • Client requests to backend

Marvel Puzzle Quest

Android | iOS | PC

Marvel Puzzle Quest is a mobile Match 3 RPG based on the Marvel universe. I worked as a generalist engineer on the Demiurge Studios team developing the game. During my time on the project we shipped a major reboot of the title which included a revamped engine, new gameplay features, almost entirely redone UI, and a new open-ended tutorial. This effort was in addition to frequent maintenance and content patches. While working on the game I implemented features across every part of the stack, including the backend, game client, engine, and platform layer. Here is a snapshot of some things I worked on:

  • Apple Sign-in
  • Deep linking support
  • Bug fixes for Android OpenGL ES rendering issues
  • Bug fixes for iOS touch input handling
  • Third party SDK upgrades
  • New UI screens
  • Re-usable UI widget library
  • New gameplay systems and features
  • New tutorial
  • Improved designer tools
  • Engine serialization system improvements
  • Unit testing harness
  • Cheat detection improvements
  • Backend instance monitoring tools
  • Bug fixes in sensitive systems such as player cloud saving and purchase receipt validation
