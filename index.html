Android | iOS

Rocket League Sideswipe is a version of the popular multiplayer car soccer title Rocket League, but reimagined for mobile. Marvel Puzzle Quest

Android | iOS | PC

Marvel Puzzle Quest is a mobile Match 3 RPG based on the Marvel universe. I worked as a generalist engineer on the Demiurge Studios team developing the game. During my time on the project we shipped a major reboot of the title which included a revamped engine, new gameplay features, almost entirely redone UI, and a new open-ended tutorial. This effort was in addition to frequent maintenance and content patches. Personal Projects

Ray Tracer in Rust (2020)

GitHub

This is a moderately complex ray tracer with a fair number of features. This project fulfilled two primary personal goals. One was to familiarize myself with Rust by building a complete program that touches on a number of different language features (architecture, performance, multithreading, etc.). The other was to construct a more mature ray tracer than my previous attempts. While there are many more features, optimizations, and refactors I would like to implement at some point, I am satisfied with the current code for “V1” of this project.

Features
  • Basic shapes
    • Sphere
    • Triangle
  • Basic materials library
    • Lambert
    • Metal
    • Dielectric
  • Custom JSON scene specification format
    • Supports reading from .obj meshes
  • Bounding volume hierarchy accelerates collision detection
    • Uses Surface Area Heuristic (SAH)
  • Diffuse lights
  • Textures
    • Solid
    • Perlin Noise
    • Images
  • Transformations allow scene manipulation
  • Basic volumes
  • Bump mapping
  • Monte Carlo importance sampling
  • Multithreaded
    • Number of threads is an optional command line argument
  • Basic stats and progress report
Wishlist
  • More shapes
    • Cylinders
    • Disks
  • Normal mapping
  • “Non-physical” lights
    • Point
    • Spot
    • Directional
  • Different texture mappings
    • Spherical
    • Cylindrical
  • True BSDF support
  • Read .mtl files
  • More robust statistics reporting

Node: Gravity Waves (2017)

Play on itch.io

Role: Lead Producer and Engineer
Awards:
  • Best Design - MassDiGI Game Challenge 2017
  • Runner-Up, College Alpha Category - MassDiGI Game Challenge 2017

Node: Gravity Waves started life as a Global Game Jam project in 2017. After the successful jam, however, a small subset of the group felt strongly about the core concept and continued development. I spearheaded this second phase, acting as lead producer and engineer. After adding new gameplay mechanics, polishing features, building menus, and ironing out bugs, Node: Gravity Waves was released on the iOS App Store. After release, lack of ongoing support unfortunately means it has been removed from distribution on the iOS App Store. But it is still available to play via browser on itch.io.
