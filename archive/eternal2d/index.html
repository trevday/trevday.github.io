Eternal2D Engine (2018)

The Eternal2D Engine is a simple 2D engine with support for a number of basic game creation features. It was developed in collaboration with Chris Juchem and Alex Zilbersher for a Game Engines course. For more in-depth information on the engine’s features, documentation of the code, and a video demonstration, please visit this page.
